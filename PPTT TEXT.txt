Title - Market Study on Modern Website Development Practices

Objective

To understand how modern websites use responsiveness, security, and interactivity to enhance

user experience and support business growth.

Description

Learners are required to conduct a brief market research study on real-world website

development practices and relate them to core web fundamentals such as HTML, CSS,

JavaScript, HTTP/HTTPS, sessions, cookies, and responsive design.

Tasks

• Research the importance of responsive web design, web security (H TTP vs HTTPS), and

interactivity

• Create a static and responsive web page using HTML and CSS

• Include standard components such as layouts and forms

• Add at least one interactive feature using JavaScript

• Develop a basic Tic-Tac-Toe game using HTML, CSS, and JavaScript

Deliverables

• Market research report (5–7 pages)

• Source code files (HTML, CSS, JavaScript)

• Screenshots or demo link (if hosted)
